# mkhook-pages

本仓库用于归档和发布多个独立的 HTML 页面。页面内容涵盖不同用途和场景，不针对任何特定域名或网站。

## 项目简介

`mkhook-pages` 是一个静态页面集合仓库，主要用于存放和托管独立的 HTML 文件。每个页面均为自包含的独立文档，不依赖外部服务或特定后端环境。适合作为轻量级的页面存档、展示或测试用途。

## 目录说明

- `pages/` — 存放所有 HTML 页面文件，按功能或时间命名
- `assets/` — 页面所需的静态资源（图片、样式、脚本等）
- `index.html` — 仓库主页，提供页面列表索引
- `README.md` — 本说明文件

## 页面归档说明

- 每个页面均为完整的 HTML 文档，可直接在浏览器中打开
- 页面之间无依赖关系，可独立运行
- 归档内容不定期更新，不保证实时同步或持续维护
- 不收集用户信息，不包含第三方跟踪代码

## 维护说明

- 本仓库由个人维护，更新频率不固定
- 欢迎通过 Issues 提交建议或报告问题
